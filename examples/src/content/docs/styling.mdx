# Styling

The package focuses on selection behavior and keeps visual customization lightweight.

## Rectangle Styling

Pass Leaflet path styling through the `options` prop.

<CodeBlock language="tsx">
{`<SelectArea
  keepRectangle
  showControl
  options={{
    color: "#2563eb",
    weight: 2,
    fillOpacity: 0.08,
  }}
/>`}
</CodeBlock>

## Control Placement

Move the built-in control by using `controlPosition`.

<CodeBlock language="tsx">
{`<SelectArea showControl controlPosition="topleft" />`}
</CodeBlock>

## App-Level Styling

For anything beyond path styling, style the surrounding app shell:

- custom toolbar buttons
- status panels
- summary cards
- route-specific layouts inside your docs or product UI

The package intentionally does not impose a design system. It only exposes enough hooks to let the rest of your application own the presentation.
