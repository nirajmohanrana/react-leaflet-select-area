# Basic Example

In this example, we'll demonstrate how to use the `SelectArea` component to select an area on a Leaflet map and log the selected bounds to the console.

To get started with `react-leaflet-select-area`, follow these steps:

1. Import the necessary components:

<CodeBlock language="javascript">
{`import { MapContainer, TileLayer } from "react-leaflet";
import { SelectArea } from "react-leaflet-select-area";
import "leaflet/dist/leaflet.css";`}
</CodeBlock>

2. Add the `SelectArea` component to your map:

<CodeBlock language="javascript" highlightedLines="11">
{`<MapContainer
      center={[51.505, -0.09]}
      zoom={13}
      className="h-full w-full"
      zoomControl={false}
    >
      <TileLayer
        url="https://{s}.tile.openstreetmap.org/{z}/{x}/{y}.png"
        attribution='&copy; <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
      />
      <SelectArea onBoundsChange={(bounds) => console.log(bounds)} />
</MapContainer>`}
</CodeBlock>

Check console for the selected bounds.
